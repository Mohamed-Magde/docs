class jina.hubapi.docker.HubIO(args)[source]¬∂

Bases: object

HubIO provides the way to interact with Jina Hub registry. You can use it with CLI to package a directory into a Jina Hub image and publish it to the world.

Examples:

  • jina hub build my_pod/ build the image

  • jina hub build my_pod/ --push build the image and push to the public registry

  • jina hub pull jinahub/pod.dummy_mwu_encoder:0.0.6 to download the image
