Bases: jina.executors.encoders.frameworks.BaseOnnxEncoder

OnnxImageEncoder encodes data from a ndarray, potentially B x (Channel x Height x Width) into a

ndarray of B x D.

Internally, OnnxImageEncoder wraps the models from onnxruntime.

Parameters

pool_strategy (str) ‚Äì the pooling strategy - None means that the output of the model will be the 4D tensor output of the last convolutional block. - mean means that global average pooling will be applied to the output of the last convolutional block, and thus the output of the model will be a 2D tensor. - max means that global max pooling will be applied.
