A simple Python client for connecting to the gateway. This class is for internal only, use the python interface jina.clients.py_client() to start PyClient if you want to use it in Python.

Assuming a Flow is ‚Äústandby‚Äù on 192.168.1.100, with port_expose at 55555.

# to search

# to index
