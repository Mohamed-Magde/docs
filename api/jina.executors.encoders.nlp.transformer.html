Automatically creates a sentence embedding from its token embeddings.

  • For BERT-like models (BERT, RoBERTa, DistillBERT, Electra ‚Ä¶) uses embedding of first token

  • For XLM and XLNet models uses embedding of last token

  • Assumes that other models are language-model like and uses embedding of last token

  • truncation_strategy (str) ‚Äì select truncation strategy. Supported values * True or ‚Äòlongest_first‚Äô (default): truncate to a max length specified in max_length or to the max acceptable input length for the model if no length is provided (max_length=None). * ‚Äòonly_first‚Äô: This will only truncate the first sequence of a pair if a pair of sequences (or a batch of pairs) is provided * ‚Äòonly_second‚Äô: This will only truncate the second sequence of a pair if a pair of sequences (or a batch of pairs) is provided * False or ‚Äòdo_not_truncate‚Äô: No truncation (i.e. can output batch with sequences length greater than the model max admissible input size)
